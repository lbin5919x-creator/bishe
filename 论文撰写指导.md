## 第2章 相关理论与技术介绍

### 2.1 交通信号控制理论基础

#### 2.1.1 交通流基本参数
- **流量（Flow）**：单位时间通过断面的车辆数
- **密度（Density）**：单位长度道路上的车辆数
- **速度（Speed）**：车辆行驶速度
- **排队长度（Queue Length）**：停车等待的车辆数
- **等待时间（Waiting Time）**：车辆在路口的累计等待时间

**代码支撑：**
```python
# environment/sumo_env.py:206-209
queue_length = traci.lane.getLastStepHaltingNumber(lane)  # 排队长度
occupancy = traci.lane.getLastStepOccupancy(lane)         # 占用率
```

#### 2.1.2 信号相位与配时
- **相位（Phase）**：一组同时获得通行权的交通流
- **绿灯时长**：本项目设置最小10秒、最大60秒
- **黄灯时长**：4秒过渡期
- **全红时长**：2秒清空期

**代码支撑：**
- [`config/settings.py`](config/settings.py:38-44) 定义了相位时长参数
- [`environment/phase_logic.py`](environment/phase_logic.py) 实现了相位状态机

#### 2.1.3 性能评价指标
- **平均等待时间**：所有车辆等待时间的平均值
- **平均排队长度**：各车道排队车辆数的平均值
- **通行量（Throughput）**：成功通过路口的车辆总数

**代码支撑：**
```python
# environment/sumo_env.py:252-266
total_wait = sum(traci.edge.getWaitingTime(edge) for edge in self.edges)
avg_queue = float(np.mean([traci.lane.getLastStepHaltingNumber(lane) for lane in self.lanes]))
```
